## Command line management

Integrating bash environment by gluing together command line artefacts like functions, aliases, completions, prompts, scripts, etc.


## Artefacts

- dotfiles
  - .profile
  - .bashrc
  - .bash_profile
  - .bash_login
  - .bash_logout
  - .inputrc
  - .dircolors
- config
  - terminal
  - globs
  - environment vars
  - env hooks
  - env handles
  - app preferences
  - locale settings
  - history settings
  - ulimit
  - mail
  - completion settings
  - shopt settings
  - shell (set) settings
- aliases
- completions
- functions
- scripts
